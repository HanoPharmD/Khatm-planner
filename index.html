How to add to your calendar
📱
iPhone / iPad
  1. Tap Download above
  2. Tap the downloaded file in your browser
  3. Tap "Add All" when prompted
  4. Opens directly in Apple Calendar ✓
🤖
Android
  1. Tap Download above
  2. Open your Files app, find the .ics file
  3. Tap the file — it should open Google Calendar
  4. Tap "Import" to confirm ✓
🖥️
Google Calendar
  1. Download the .ics file
  2. Go to calendar.google.com
  3. Click ⚙️ Settings → Import & Export
  4. Upload the file and click Import ✓
💡 Tip: On iPhone, make sure Safari is your browser — it handles .ics files best.
